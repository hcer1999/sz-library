# 三站图书馆

## 项目简介

三站图书馆是一个基于 nextjs 的图鉴系统，可以为用户查询三国志·战略版武将的属性、战法、兵书、阵容搭配等。

一个现代化的站点，UI 需要简洁，功能需要强大，并且要兼容移动端的展示。

## 项目架构（本项目只包含前端）

- 前端：nextjs
- 后端：nodejs
- 数据库：mysql

## 项目功能

- 查询武将属性、战法、兵书、阵容搭配等
- 查询武将战法效果
- 查询武将阵容搭配
- 查询武将兵书搭配
- 查询武将属性
- 查询武将战法
- 查询武将兵书
- 查询武将阵容

## 项目前端技术栈

- nextjs
- react
- typescript
- tailwindcss
- shadcn/ui
